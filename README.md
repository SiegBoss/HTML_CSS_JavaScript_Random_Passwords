# PassGen - Generador de Contraseñas Web

Este proyecto es un generador de contraseñas seguras implementado con tecnologías web (HTML, CSS y JavaScript) que cuenta con una interfaz de usuario moderna y atractiva basada en Neumorfismo. Permite generar claves robustas y personalizadas directamente en el navegador.

## Características

- **Generación Personalizable**: Genera contraseñas aleatorias ajustando la longitud y los tipos de caracteres (Mayúsculas, Minúsculas, Números y los símbolos seguros `@#$%`).
- **Palabra Semilla (Opcional)**: Permite ingresar una palabra base y aplicarle un cifrado visual ("leet speak") mezclado con caracteres aleatorios.
- **Medidor de Fuerza**: Incluye una barra visual dinámica que cambia de color para indicar el nivel de seguridad de la contraseña generada.
- **Copia Rápida**: Un botón intuitivo de un solo clic para copiar la contraseña al portapapeles con feedback visual interactivo.

## Estructura del Proyecto

El proyecto está modularizado en los siguientes archivos para un mejor mantenimiento:

- `index.html`: Contiene la estructura semántica de la aplicación web.
- `style.css`: Archivo de estilos con diseño responsive, paleta de colores moderna y transiciones fluidas.
- `script.js`: Archivo principal con la lógica de generación, validación y control de la interfaz de usuario.

## Requisitos

Para ejecutar este proyecto, no necesitas instalar ningún software o lenguaje en tu sistema. Solo necesitas abrir el archivo `index.html` en cualquier **Navegador Web Moderno** (Chrome, Firefox, Safari, Edge).

# PassGen - Web Password Generator

This project is a secure password generator implemented with web technologies (HTML, CSS, and JavaScript) featuring a modern and attractive Neumorphic user interface. It allows you to generate robust and customized passwords directly in the browser.

## Features

- **Customizable Generation**: Generate random passwords by adjusting the length and character types (Uppercase, Lowercase, Numbers, and safe symbols `@#$%`).
- **Seed Word (Optional)**: Allows you to input a base word and apply a visual cipher ("leet speak") mixed with random characters.
- **Strength Meter**: Includes a dynamic visual bar that changes color to indicate the security level of the generated password.
- **Quick Copy**: An intuitive one-click button to copy the password to the clipboard with interactive visual feedback.

## Project Structure

The project is modularized into the following files for better maintainability:

- `index.html`: Contains the semantic structure of the web application.
- `style.css`: Stylesheet with responsive design, a modern color palette, and smooth transitions.
- `script.js`: Main file containing the generation logic, validation, and UI control.

## Requirements

To run this project, you do not need to install any software or languages on your system. You just need to open the `index.html` file in any **Modern Web Browser** (Chrome, Firefox, Safari, Edge).
